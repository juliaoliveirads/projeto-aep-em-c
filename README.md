# projeto-aep-em-c
Sistema de Triagem de Sintomas para Comunidades Rurais

Este projeto foi desenvolvido como parte da AEP 2025 do curso de Engenharia de Software.
A proposta é criar um sistema simples, feito em linguagem C, que ajude na triagem básica de sintomas em comunidades rurais ou de baixa renda, principalmente em locais onde não há acesso fácil à internet.

Objetivo:

O objetivo principal é facilitar o primeiro atendimento, permitindo que o usuário informe alguns sintomas e receba uma orientação inicial sobre o que pode estar sentindo e se deve procurar um posto de saúde.
A ideia é que o sistema funcione de forma rápida, offline e com baixo consumo de recursos, podendo ser usado em computadores simples.

Com este projeto, espera-se:

-> Tornar o processo de triagem mais acessível para quem vive longe de centros urbanos;
-> Aumentar a conscientização sobre sintomas comuns e possíveis cuidados;
-> Mostrar como a tecnologia pode ser usada como uma ferramenta de apoio social.
